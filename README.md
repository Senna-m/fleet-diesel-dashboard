# 🚀 Assistente Trafego

Um sistema que ajuda profissionais de logística voltados para turismo a escolher o melhor veículo para determinada rota, utilizando como base o histórico de abastecimento e rotas anteriores.

---

## 📖 Sobre o Projeto

O **Assistente Trafego** é uma solução desenvolvida para apoiar a tomada de decisão na área de transporte turístico.

O sistema analisa dados históricos de:

- Consumo de combustível
- Rotas já realizadas
- Distância percorrida
- Desempenho dos veículos

Com essas informações, o sistema recomenda o veículo mais adequado para uma nova rota, buscando:

- ⛽ Redução de custos com combustível  
- 📉 Otimização de consumo  
- 🚐 Melhor aproveitamento da frota  
- 📊 Tomada de decisão baseada em dados  

Este projeto foi desenvolvido como estudo aplicado em Ciência de Dados e Backend.
